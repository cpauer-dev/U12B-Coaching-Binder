# Touchline Binder PWA v0.1

A local-first, installable youth soccer coaching app.

## What "PWA" means
A Progressive Web App is a website that can be installed on your phone and opened like a normal app. Once loaded, this project works offline and stores its data on the device.

## Included
- Preloaded BSSC U12 Boys B roster
- Jersey numbers and conflict warnings
- Parent contact fields
- Official-registration and TeamReach status
- Incremental events for practices, fall/spring league, indoor sessions, tournaments, and scrimmages
- Attendance
- Practice plans and drill library
- Match results and After Action Reviews
- Player-development notes
- Equipment inventory
- Local JSON backup/export and restore/import

## Important
Opening index.html directly from a file manager will show the app, but install/offline features normally require the files to be served over HTTPS.

## Easiest ways to install
1. Upload this folder to a static host such as GitHub Pages, Netlify, Cloudflare Pages, or Firebase Hosting.
2. Open the HTTPS address in Chrome on Android.
3. Choose "Install app" or "Add to Home screen."

No backend or account is required. Data remains local to each device unless you export a backup.
